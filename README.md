# Sara Invernadero

## Descripción del Proyecto

Sara Invernadero es una página web dedicada a la venta de plantas, contratar servicios de jardinería y diseño de jardines, la página permite comprar flores, conocer servicios especializados, visualizar propuestas de diseño y contactar al negocio mediante un formulario.

## Temática Elegida

Venta de plantas, jardinería y diseño de jardines para hogares y espacios verdes.

## Paleta de Colores

-Amarillo Suave: #FAF064

-Georgia
-Serif 

## Instrucciones para usar el filtro

1. Ingresar una palabra relacionada con la planta.
2. También es posible buscar por categoría:
    -Sol
    -Sombra
    -Luz indirecta
3. Presionar el botón Buscar o la tecla Enter.
4. El sistema mostrará únicamente las plantas que coincidan con el criterio ingresado.
